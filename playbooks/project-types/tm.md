# T&M Project Type Playbook

## Use When

Use this playbook for time-and-materials projects where scope can evolve and the client pays for actual effort.

## Primary Controls

- burn rate
- budget consumed
- forecast to next milestone
- capacity allocation
- utilization
- backlog priority
- unplanned work
- timesheet variance

## Agent Rules

- Focus on transparency, prioritization, and value delivered per burn.
- Flag increasing burn without visible delivery progress.
- Convert vague scope additions into prioritized backlog items.
- Recommend trade-offs instead of treating all requested work as equally urgent.
- Separate budget impact from delivery risk.

## Required Reporting

- consumed effort
- remaining forecast
- priority changes
- blockers
- value delivered
- budget risk

## Escalation Triggers

- budget burn exceeds forecast
- team is busy but outcomes are unclear
- client requests new work without reprioritization
- recurring blockers reduce utilization
