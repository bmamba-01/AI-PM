# AI-PM Toolkit T&M Test Project

## Purpose

Use AI-PM Toolkit to manage and verify AI-PM Toolkit itself as a T&M test project.

## Delivery Target

- Start date: 2026-06-21
- Completion target: 2026-06-28
- Commercial model: Time and Materials
- Methodology: Scrum
- Active human role: Project Manager only

## Communication

- Primary channel: Discord
- Bot/adaptor: Hermes Agent Bot
- Server mode: local laptop server
- External Discord messages require approval before sending.

## Tracking

- Primary tracking system: Notion
- Current automation status: local import files prepared
- Notion import CSV: `integrations/notion/issues.csv`
- Notion schema guide: `integrations/notion/notion-database-schema.md`

## Acceptance Criteria

- Toolkit can initialize/adopt a project.
- CLI JSON commands remain parseable by agents.
- Desktop setup gateway is smoke-tested.
- Daily and weekly workflows run local-first.
- Issues/features are tracked in Notion or import-ready files until Notion MCP is connected.
- Discord/Hermes bot can query status through local server without direct mutation.
