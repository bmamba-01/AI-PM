# Meeting Agenda Template

## Purpose
Provide a structured framework for planning and conducting effective project meetings. Ensures all participants are prepared, time is used efficiently, and decisions/actions are captured.

## When to Use
- Weekly team sync meetings
- Sprint planning sessions
- Stakeholder reviews
- Risk/issue triage meetings
- Retrospectives
- Any scheduled project meeting

## Required Inputs
| Input | Source | Required |
|-------|--------|----------|
| Meeting title and type | Meeting organizer | Yes |
| Date, time, duration | Calendar invite | Yes |
| Attendees (required/optional) | Project roster | Yes |
| Previous meeting action items | Prior minutes | If recurring |
| Current sprint/iteration goals | Sprint board | For sprint meetings |
| Open risks/issues to discuss | Risk register, issue tracker | As needed |
| Decisions needed | Product owner, tech lead | As needed |
